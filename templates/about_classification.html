About the Dataset: Titanic Dataset

This page provides information about the Titanic dataset used in the project.

Source

The dataset for classification analysis was obtained from Kaggle. The dataset includes a CSV file named titanic-train.csv, which contains information about passengers aboard the Titanic.

Source: Kaggle (link: Titanic Dataset)

Data File: titanic-train.csv

Variable Definitions

  • PassengerId: Unique identifier for each passenger.
  • Pclass: Ticket class (1st, 2nd, 3rd).
  • Name: Passenger's name.
  • Sex: Passenger's gender.
  • Age: Passenger's age in years.
  • SibSp: Number of siblings/spouses aboard the Titanic.
  • Parch: Number of parents/children aboard the Titanic.
  • Ticket: Ticket number.
  • Fare: Passenger fare.
  • Cabin: Cabin number.
  • Embarked: Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton).
