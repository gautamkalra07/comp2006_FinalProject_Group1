About House Prices Regression Project

Source: Kaggle (link: House Prices Dataset)

The project uses the House Prices dataset obtained from Kaggle. This dataset contains information about various features of houses and their corresponding sale prices.

The project covers the following steps:

  • Data Loading and Preprocessing: Loading the dataset into a Pandas DataFrame, handling missing values, encoding categorical variables, and scaling numerical variables.
  • Data Splitting: Splitting the data into training, validation, and testing sets using the train_test_split function from scikit-learn.
  • Model Building and Hyperparameter Tuning: Building Random Forest, Gradient Boosting, and K Neighbors Regression models using scikit-learn. Hyperparameters are tuned using GridSearchCV.
  • Model Evaluation: Evaluating the performance of the models on the validation set using the Root Mean Squared Error (RMSE) metric.
  • Model Training on Combined Data: Retraining the best models on the combined training and validation data.
  • Model Evaluation on Test Set: Evaluating the retrained models on the test set and calculating the RMSE for each model.
  • Model Saving: Saving the trained models for future use.

By the end of this project, you will have a good understanding of how to build, evaluate, and optimize regression models using Python and scikit-learn.
